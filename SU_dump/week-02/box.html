Hi, ich bin ein Box Model!

  • rot = content
  • grün = padding
  • schwarz = border
  • nicht sichtbar: margin (Abstand nach dem border)

Diese Bereiche können auch über die Developer-Tools inspiziert werden.

Display Varianten

block

display block (z.B. ein p )

ich bin display block mit height: 3em und width: 80%

  • nimmt die komplette verfügbare Breite ein
  • hat Zeilenumbruch am Anfang und am Ende des Elements
  • Größe veränderbar

inline

display inline (z.B. ein q )
ich bin ein inline Element ich bin ein weiteres inline Element, direkt dahinter
ich bin ein inline Element mit height: 3em und width: 80%, aber das bewirkt nichts...
  • nimmt nur den für den content benötigten Platz ein
  • hat keinen Zeilenumbruch
  • Größe lässt sich nicht manipulieren

inline-block

ich bin inline-block, das mit der Größe klappt jetzt ich bin auch inline-block
  • nimmt für den content benötigte Breite ein, außer man spezifiziert andere Größe
  • hat keinen Zeilenumbruch

Weitere Unterschiede zwischen block , inline und inline-block werden bei der Verwendung von padding und margin deutlich:

Ich bin ein block Element ich bin inline mit padding links und rechts

Ich bin ein block Element ich bin inline mit margin links und rechts

Ich bin auch block

Ich bin ein block Element ich bin inline mit vertikalem margin/padding. Ich "lege" mich über andere Elemente.

Ich bin auch block

Ich bin ein block Element ich bin inline mit vertikalem margin/padding. Ich beanspruche Platz für mich.
